A simple command-line based 
**Student Management System** built using Python.
and All tasks are stored using a **TXT file (students.txt)**.

the file which includes:

-   Add New Students
-   View Students
-   Search Student
-   Update Student
-   Delete Student

# Project Structure
      Project folder/
         main.py
         students.txt
         README.md
        
## How to Run
1.  Install Python (3.15 recommended)
2.  Save all project files in one folder
3.  Run the script:

``` bash
python main.py
```
The application will automatically create **students.txt** on the first
run.
## Features in this Project
### 1. Add Student
Adds a new student to the TXT.
### 2. Display All Students
Displays all students.
### 3. Search Student
Search and display only the selected student.
### 4. Update Student
search and update the selected student
### 5. Delete Student
delete the selected student
